# QFX Converter

Converts a set of OFX (Open Financial Exchange) and QFX (Intuit's Quicken) files into a single CSV (Comma Separated Values) file for further processing

## To Build & Run
```
swift build
.build/debug/QfxConverter <folder>
```

## To Use

```
./QfxConverter <folder>
```
